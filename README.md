This program allows users to estimate the range of heavy charged particles (proton, alpha, or carbon) in an arbitrary material using the continuous slowing down approximation (CSDA). The user must provide the following material parameters: material density (g/cm^3), material Z/A, and mean excitation energy I (MeV). The incident heavy charged particle parameters can be changed such as the initial kinetic energy (MeV) and the step size (cm). The results are in good agreement with NIST values albeit with some fluctuation and are recommended for estimation or screening purposes only. 

Future editions may include built-in methods of calculating Z/A and I for more complex materials, such as novel compounds or polymers. More corrections may be included to make the values more physically accurate. UI changes may be implemented if time allows. For now view my profile for any calculator updates for relevant parameters.

Note that the Bethe stopping power formula is used but it is not reliable at low energies (~ < 1 MeV).
